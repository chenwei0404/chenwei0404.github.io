基本需求

下载node.js（node.js官网下载 https://nodejs.org/en）下载完成后使用node -v 和 npm -v可以查看版本。如果从事的工作与nodejs相关介意下载nvm,然后通过nvm下载node。

介意: 下载完npm后通过npm install -g cnpm –registry=http://registry.npm.taobao.org下载cnpm依赖。切换npm为淘宝镜像下载速度会比npm快一些
