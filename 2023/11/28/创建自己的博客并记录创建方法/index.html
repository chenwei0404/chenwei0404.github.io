# 添加 README.md 渲染格式

一般 github 项目的 README.md 都是对项目的自述和介绍等，但 hexo 创建部署的时候并没有给我们创建可以手动在 \themes\hexo-theme-next\source 下自行创建一个 README.md。但部署到 git 上时会发现 README.md 变成了 README.html，被自动转格式了，内容也会有一部分变成乱码就需要在博客的_config.yml 中配置
